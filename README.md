# ✋✌️ Jokenpô Online

Jogo clássico de Pedra, Papel e Tesoura desenvolvido com HTML, CSS e JavaScript.  
Projeto criado para treinar lógica de programação, manipulação do DOM e interação com o usuário.

Treinando lógica, relaxando a mente.

## Funcionalidades

- Escolha entre pedra, papel ou tesoura
- Jogada automática do computador
- Placar de pontos do usuário e do computador
- Mensagens de vitória, derrota e empate com cores
- Botão para reiniciar o jogo
- Layout centralizado e responsivo

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
